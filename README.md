## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+
- npm, yarn, or pnpm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/jimmyxu07/nano-banana-clone.git
cd nano-banana-clone
```

2. Install dependencies:
```bash
npm install
# or
npm install --legacy-peer-deps  # Recommended to resolve dependency conflicts
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
